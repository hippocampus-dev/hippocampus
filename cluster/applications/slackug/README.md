slackug is a Slack bot that manages user groups within Slack channels.

## Development

```sh
$ export SLACK_BOT_TOKEN=<YOUR SLACK BOT TOKEN>
$ export SLACK_SIGNING_SECRET=<YOUR SLACK SIGNING SECRET>
$ make dev
```
