## Project Structure

```
.
├── packages/                                # Rust workspace with 27 core libraries
│   ├── hippocampus-core/                   # Core: Full-text search library with indexing, tokenization, and ranking
│   ├── hippocampus-server/                 # Core: gRPC/HTTP server with OpenTelemetry and Prometheus metrics
│   ├── hippocampus-client/                 # Core: gRPC client library for service-to-service communication
│   ├── hippocampus-standalone/             # Core: Standalone application binary
│   ├── hippocampus-web/                    # Core: Web interface components
│   ├── hippocampus-configuration/          # Core: Runtime configuration management
│   ├── hippocampusql/                      # Core: SQL query interface
│   ├── jwt/                                # Auth: JWT authentication and validation utilities
│   ├── jwt_derive/                         # Auth: JWT derive macros for automatic implementation
│   ├── opentelemetry-tracing/              # Observability: Distributed tracing integration
│   ├── elapsed/                            # Observability: Elapsed time measurement utilities
│   ├── elapsed_macro/                      # Observability: Macro support for elapsed time tracking
│   ├── retry/                              # Resilience: Request retry logic with exponential backoff
│   ├── hedged/                             # Resilience: Hedged request patterns to reduce tail latency
│   ├── gcs/                                # Storage: Google Cloud Storage integration
│   ├── serde_binary/                       # Storage: Binary serialization utilities
│   ├── bloom-filter/                       # Storage: Probabilistic data structure for membership testing
│   ├── mysql-protocol-parser/              # Protocol: MySQL wire protocol parser
│   ├── elf/                                # Protocol: ELF binary format parser
│   ├── openai/                             # Integration: OpenAI API client library
│   ├── audio/                              # Utility: Audio processing utilities
│   ├── enum_derive/                        # Utility: Enum derive macros
│   ├── error/                              # Utility: Error handling utilities
│   ├── husky/                              # Utility: Git hooks automation
│   ├── bakery/                             # Utility: Build and packaging utilities
│   └── hippocampus-wasm-tokenizer-example/ # Example: WASM tokenizer demonstration
│
├── cluster/                                 # Kubernetes platform infrastructure
│   ├── applications/                       # 66 microservices categorized by function
│   │   ├── embedding-gateway/              # AI/ML: S3-cached OpenAI embeddings proxy
│   │   ├── embedding-retrieval/            # AI/ML: Vector similarity search service
│   │   ├── embedding-retrieval-loader/     # AI/ML: Vector data loader
│   │   ├── whisper-worker/                 # AI/ML: Speech-to-text using Whisper
│   │   ├── translator/                     # AI/ML: Machine translation service
│   │   ├── rust_de_llama/                  # AI/ML: LLM inference server
```
